PROJECT TITLE : PRIYANSHI EXPENSE TRACKER

MOTIVATION: It is challenging to manually manage daily expenses, which frequently leads to uncertainty about where money is being spent.  Using Python and Streamlit, this project offers an easy-to-use digital solution for monitoring and evaluating spending.  By keeping track of expenses and offering visual analytics, it aids users in maintaining financial discipline.



DATA DESCRIPTION:  Expense records for the project are kept in a CSV file called expenses.csv.  The following columns are included in the dataset:  Description of Column Name  Date  Category  Date of expenditure  Expense category (Food, Transportation, Entertainment, Utilities, Other)  Amount  Spending money  An explanation  A brief note regarding  cost  When the user saves expenses from the application, the file is generated automatically.



FEATURES:  Add fresh entries for expenses  Save costs in CSV.  Save money by loading  Show every expense that has been recorded in a tabular format.  Use a bar chart to see your spending.  Streamlit UI is clear and uncomplicated. 



TECHNOLOGIES: Tools and Technologies Employed  3.12 Python,  Streamlit  ,Pandas,  Seaborn and Matplotlib  Visual Studio Code.


INSTRUCTIONS FOR USAGE AND INSTALLATION:  Install the necessary libraries in step one.  Copy the code  Install streamlit pandas matplotlib seaborn using pip.  Step 2: Launch the Streamlit Program  run app.py in streamlit.  Step 3:  Use the sidebar to:  Increase costs  Save the file.  Open the file  Make statistics visible.

GUIDELINES FOR TESTING: 

1. Use Ul to add sample costs  Third.
2. Reduce costs and confirm the creation of CSV.
3. Close the application, then reopen it and select Load Expenses. 
4. To view the graph output, click Visualize Expenses.
