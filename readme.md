# Personal Finance Manager

A simple command-line based Personal Finance Manager that allows users to record, track, and analyze their daily financial transactions.  
This project is designed as part of the VITyarthi “Build Your Own Project” requirement and demonstrates concepts such as modular programming, file handling, data processing, and user-interactive workflows in Python.

---

##  Features

### **1. Add Transactions**
- Add income or expense records with date, description, and amount.

### **2. View Transactions**
- View all transactions or filter by date range.

### **3. Summary Report**
- Total income  
- Total expenses  
- Net savings

### **4. Search Function**
- Search transactions by description or amount.

### **5. File Handling**
- Save all transactions to a JSON file  
- Load existing data from a JSON file

### **6. User-friendly CLI**
- Clear menu-driven interface  
- Input validation and error messages

---

##  Technologies Used
- **Python 3**
- **JSON** for data storage
- Modular programming (`finance_utils.py`, `main.py`)
